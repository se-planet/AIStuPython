# 第二章 · 变量与数据类型

## 2.1 变量核心概念

- **变量本身没有类型**，类型属于变量指向的数据
- 变量是"便利贴"，可以随时贴到不同类型的数据上
- `type(x)` 查的是数据，不是变量
- `id(x)` 查看内存地址

## 2.2 基本数据类型

| 类型 | 关键字 | 示例 | 特殊点 |
|------|--------|------|--------|
| 整型 | `int` | `1, -1, 200` | 0b/0o/0x 前缀表示不同进制 |
| 浮点 | `float` | `1.1, -4.5` | 科学计数法 `5.12e2` |
| 布尔 | `bool` | `True, False` | 首字母大写！ |
| 字符串 | `str` | `"hello"` | 单/双/三引号都行 |
| 空值 | `None` | `None` | 首字母大写！ |

### 假值全家桶
`0` `0.0` `""` `[]` `{}` `()` `set()` `None` `False`

## 2.3 格式化输出

```python
# 方式1：% 占位符
"我叫%s，今年%d岁" % ("小明", 18)

# 方式2：format()
"我叫{}，今年{}岁".format("小明", 18)

# 方式3：f-string（推荐！）
f"我叫{name}，今年{age}岁"
f"计算结果：{5 + 3}"         # {} 内可以放表达式！
```

| 占位符 | 含义 | 修饰示例 |
|--------|------|----------|
| `%s` | 字符串 | — |
| `%d` | 整数 | `%03d` 宽度3，0填充 |
| `%f` | 浮点数 | `%.2f` 保留两位小数 |
| `%%` | 百分号 | — |

## 2.4 类型转换

| 函数 | 作用 | 示例 |
|------|------|------|
| `int(x)` | 转整数 | `int(3.9)` → `3`（截断） |
| `float(x)` | 转浮点 | `float("12.3")` → `12.3` |
| `str(x)` | 转字符串 | `str(100)` → `"100"` |
| `round(x, n)` | 四舍五入 | `round(3.141, 2)` → `3.14` |

- **隐式转换**：`4 + 1.2` → `5.2`（float），向高精度自动转
- **显式转换**：调用上述函数主动转换
- `"123" + 18` → **报错**！Python 不会自动拼接 str 和 int

## 2.5 变量命名规则

- 字母/下划线开头 + 字母/数字/下划线
- 不能是关键字（`class` `if` `for` 等）
- 区分大小写（`name` ≠ `Name`）
- 推荐：`my_name`（蛇形命名）或 `myName`（驼峰命名）

## 2.6 字符串驻留机制

- [-5, 256] 的整数被缓存
- 纯字母数字下划线的字符串被缓存
- `is` 判断是否同一对象，`==` 判断值是否相等

## 4.1 顺序控制

代码从上到下逐行执行，最基本的执行方式。

### if-elif-else

```python
if 条件1:
    语句块1

    语句块2
```

- 条件可以是任意值，Python 自动转布尔

### 嵌套分支

```python

    if 条件B:
        ...
    else:
        ...
else:
    ...
```

## 4.3 循环控制

### for 循环

```python
for 变量 in 可迭代对象:
    语句块
else:           # 正常结束才执行
    语句块
```

### range() 详解

```python
range(stop)              # 0 到 stop-1
range(start, stop)       # start 到 stop-1
range(start, stop, step) # start 到 stop-1，步长 step
```

```python
list(range(5))          # [0, 1, 2, 3, 4]
list(range(2, 6))       # [2, 3, 4, 5]
list(range(1, 10, 3))   # [1, 4, 7]
list(range(10, 0, -1))  # [10, 9, ..., 1]  倒序
```

### while 循环

```python
while 条件:
    语句块
else:           # 条件变 False 正常结束才执行
    语句块
```

## 4.4 循环控制关键字

| 关键字 | 行为 |
|--------|------|
| `break` | 立即**退出整个循环**，跳过 else |

| `return` | 退出函数（也退出循环） |

```python

    if i == 1:

    print("正常结束") # 不会被打印
```

## 4.5 嵌套循环

- 外层循环一轮，内层循环跑满全部

# 第五章 · 函数

## 5.1 函数定义与调用

```python
def 函数名(参数列表):
    """文档字符串"""
    函数体
    return 返回值    # 可省略，默认返回 None
```

- 函数必须先定义后调用
- 可以多次调用
- 函数名重复定义会覆盖之前的

## 5.2 返回值

```python
def f():
    return 1, 2      # 返回元组 (1, 2)

def g():
    print("hello")   # 没有 return → 返回 None
```

- `return` 可返回多个值（打包为**元组**）
- 无 `return` 时默认返回 `None`
- `return` 立即退出函数

## 5.3 参数类型

| 参数类型 | 语法 | 示例 |
|----------|------|------|
| 位置参数 | `(a, b)` | `f(1, 2)` |
| 默认值参数 | `(a, b=10)` | `f(1)` 或 `f(1, 20)` |
| 可变位置 | `(*args)` | 打包为**元组** |
| 可变关键字 | `(**kwargs)` | 打包为**字典** |

> ⚠️ 默认值参数必须放在非默认值参数的**后面**

### *args 示例

```python
def sum_all(*args):
    print(type(args))   # <class 'tuple'>
    for n in args:
        print(n)

sum_all(1, 2, 3)   # args = (1, 2, 3)
```

### **kwargs 示例

```python
def info(**kwargs):
    print(type(kwargs))   # <class 'dict'>
    for k, v in kwargs.items():
        print(k, v)

info(name="小明", age=18)   # {'name': '小明', 'age': 18}
```

## 5.4 作用域

### 核心规则

```python
x = 10       # 全局变量

def f():
    print(x) # ✅ 只读→自动找外层

def g():
    x = 20   # 创建新的局部变量 x
    print(x) # 20（局部覆盖全局）

def h():
    global x # 声明使用全局 x
    x = 30   # 修改全局 x
```

| 场景 | 行为 |
|------|------|
| 函数内**只读** | 自动找外层变量 |
| 函数内**赋值** | 创建局部变量 |
| 函数内赋值+读 | ❌ UnboundLocalError |
| 用 `global` 声明 | 操作全局变量 |

### 闭包（Closure）

```python
def outer():
    x = 5
    def inner():
        print(x)     # inner 记住了 outer 的 x
    return inner

f = outer()    # f = inner 函数
f()            # 输出 5（x 还活着）
```

> 闭包 = 函数 + 它记住的外部变量。即使外层函数执行结束，内层仍能访问。

## 5.5 递归

```python
def factorial(n):
    if n == 1:
        return 1         # 递归出口
    return factorial(n-1) * n

factorial(4)  # 24
```

两个要素：**递归出口** + **向出口逼近**

## 5.6 lambda 匿名函数

```python
lambda 参数: 表达式       # 只能单行！返回表达式结果

add = lambda a, b: a + b
add(3, 5)  # 8
```

- 只能写**单行表达式**，不能包含语句块
- 适合做简单函数的参数传递

## 5.7 函数作为参数传递

```python
def apply(func, x, y):
    return func(x, y)

apply(lambda a, b: a + b, 10, 20)   # 30
apply(max, 10, 20)                   # 20
```

# 第七章 · 排序和查找

## 7.1 冒泡排序

**核心思想：** 重复比较相邻元素，按规则交换，每轮把最大/最小的"冒"到最后。

```python
def bubble(arr):
    n = len(arr)
    for i in range(n - 1):           # 外层：n-1 轮
        for j in range(n - 1 - i):   # 内层：每轮少比一个
            if arr[j] > arr[j + 1]:
                arr[j], arr[j + 1] = arr[j + 1], arr[j]
    return arr
```

| 参数 | 说明 |
|------|------|
| 外层 `n-1` 轮 | n个元素排 n-1 轮即可 |
| 内层 `n-1-i` | 每轮尾部已排好，无需再比较 |
| 时间复杂度 | O(n²) |

## 7.2 顺序查找

逐个遍历直到找到目标，最坏查 n 次。不要求有序。

```python
def linear_search(arr, target):
    for i in range(len(arr)):
        if arr[i] == target:
            return i
    return -1
```

## 7.3 二分查找

**前提：数组必须已排序。** 每次从中间砍掉一半。

```python
def binary_search(arr, target):
    left, right = 0, len(arr) - 1
    while left <= right:
        mid = (left + right) // 2
        if arr[mid] == target:
            return mid
        elif arr[mid] < target:
            left = mid + 1     # 目标在右边
        else:
            right = mid - 1    # 目标在左边
    return -1
```

| | 顺序查找 | 二分查找 |
|------|:--:|:--:|
| 是否要求有序 | ❌ | ✅ |
| 时间复杂度 | O(n) | O(log n) |
| 适用场景 | 小数据/无序 | 大数据/有序 |

### 二分查找找不到的示例：`[1,3,5,7,9]` 找 `4`

```
mid=2(arr[2]=5)>4 → right=1
mid=0(arr[0]=1)<4 → left=1
mid=1(arr[1]=3)<4 → left=2
left=2 > right=1 → 区间为空 → -1
```

# 第八章 · 断点调试

## 8.1 断点（Breakpoint）

让程序暂停在指定行，查看变量值、单步追踪执行流程。

## 8.2 核心快捷键

| 快捷键 | 功能 | 说明 |
|--------|------|------|
| `F7` | Step Into | 进入函数内部 |

|------|------|

| 进入函数内部 | F7 Step Into |
| 跳过不关心区域 | F9 跳到下一个断点 |

> 断点调试是实操技能，建议在 IDE 中亲手打断点练习。

# 第九章 · 模块和包

## 9.1 import 导入

```python
import math              # 导入整个模块
import random as r       # 起别名（别名不改功能）
from math import sqrt    # 只导入一个函数
from math import *       # 导入全部（❌ 不推荐，污染命名空间）
```

| 写法 | 调用方式 | 推荐 |
