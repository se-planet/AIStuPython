## ❌ Q1 · \r 回车符

```python
print("Hello,\rWorld")   # 输出 → World,
```

`\r` 回到行首但不换行，World 覆盖 Hello，逗号保留。

## ❌ Q4 · type(print)

```python
type(print)   # <class 'builtin_function_or_method'>
```

## ❌ Q5 · 负数整除（向下取整）


## ❌ Q6 · 负数取模


## ❌ Q11 · 可变默认参数陷阱

```python
def f(a, b=[]):    # b=[] 只在定义时执行一次！
    b.append(a)
    return b

f(1)  # [1]
f(2)  # [1, 2] ← 同一个列表，没重置！
```

- `b=[]` 在 `def` 时创建，以后每次调用共用这个列表
- ✅ 正确做法：`b=None` + 函数内判断创建

```python
def f(a, b=None):
    if b is None:
        b = []
    b.append(a)
    return b
```

> **看家规则：默认参数永远不用可变类型 `[]` `{}` `set()`，用 `None` 替代。**

## ❌ Q1 · 不可变类型赋值（整数）

```python
a = 10
b = a       # b 也指向 10
a = 20      # a 换新目标，b 不变
print(b)    # 10，不是 20！
```

整数是**不可变类型**，`a = 20` 是让 a 指向新对象，不修改原对象。

> 对比：列表 `b[0]=100` 直接修改内容，所有引用都受影响。

## ❌ Q3 · 变量命名规则

```python
1name     # ❌ 不能数字开头
my-name   # ❌ - 是减号，只允许字母/数字/_
_class    # ✅ 下划线开头合法
class     # ❌ Python 关键字
```

> 命名 = (字母|下划线) + (字母|数字|下划线)*，不能是关键字。

## ❌ Q9 · while-break-else

```python
i = 5
while i > 0:
    i -= 1
    if i == 3:
        break       # 中断循环
else:
    print("正常结束")  # ❌ break 后不会执行！
print("i =", i)       # ✅ 执行 → i = 3
```

**`break` 退出循环后，`else` 块被跳过。** 只有循环自然结束（条件变 False）才会执行 `else`。

## ❌ Q1 · 无 return 返回 None

```python
def greet():
    print("你好")      # 没有 return

result = greet()        # → 打印 "你好"
print(result)           # → None（无 return 自动返回 None）
```

函数没有 `return` 时默认返回 `None`。`print(result)` 会打印出 `None`，完整输出两行。

## ❌ Q3 · print 多参数用空格分隔

```python
x, y = 15, 5
print(x, y)     # → 15 5（空格分隔，不是 15,5）
```

`print` 默认 `sep=' '`，多个参数用空格隔开。要逗号分隔需：`print(f"{x},{y}")`

---

## ❌ Q7 · 局部变量不可与外部同名变量混合使用

```python
x = 10
def change():
    x = x + 1     # ❌ UnboundLocalError！
```

Python 看到函数内 `x =` 就认为 x 是局部变量，但 `x+1` 需要先读值，此时局部 x 还没赋值 → 崩溃。

```python
def change():
    global x      # ✅ 声明使用外部 x
    x = x + 1
```

> **规则**：函数内**只读不写**→自动找外层；**赋值**→创建局部变量。

## ❌ Q9 · lambda 只能单行表达式

```python
add = lambda a, b: a + b    # ✅ 单行表达式

# ❌ 不能多行
lambda a, b:
    return a + b
```

lambda 是匿名函数，只能放 `return 表达式` 的简写形式，不能包含语句块。
