## 🧠 学习方法

每个主题三步走，不多不少：

1. **AI 讲** — 概念 + 示例代码 + 复杂度分析
2. **我动手** — 逐行理解 → 改参数实验 → 做练习题
3. **AI review** — 检查代码 → 纠正思路 → 记错题
